# EF Subsystem
CZ3003 Emergency Force Subsystem Simulation

## Dependencies

- Python 3
- Flask
- flask-restful

You can install python dependencies with `pip3 install requirements.txt`

## Run

### *nix

- Run `app.py` directly after `chmod +x` it, or run `python3 app.py` from terminal

### Windows

- Right click `app.py`
- Select `open with`
- Select `python 3`
